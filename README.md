# Device Management System

## 📌 Project Overview

The Device Management System is a web-based application developed using PHP and MySQL to manage devices efficiently. The system allows users to register, log in, view available devices, and start sessions, while administrators can manage devices, monitor sessions, and control user access through a secure admin panel.

---

## ✨ Features

### User Features
- User Registration
- User Login
- Forgot Password
- Dashboard
- View Device Status
- Start Device Session
- End Session
- Profile Management

### Admin Features
- Admin Login
- Dashboard with Statistics
- Add New Device
- Edit Device Details
- Delete Device
- View Registered Users
- Monitor Active Sessions
- Manage Device Status

## ⚙ Installation

### Step 1

Clone or download the project.

### Step 2

Copy the project folder into:

```
xampp/htdocs/
```

### Step 3

Start:

- Apache
- MySQL

from the XAMPP Control Panel.

### Step 4

Open phpMyAdmin and create a database:

```
device_management
```

### Step 5

Import the SQL file into the database.

### Step 6

Update the database configuration if required.

### Step 7

Open your browser and visit:

```
http://localhost/device-management
```
